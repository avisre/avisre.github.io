# avisre.github.io

Personal portfolio site for **Avinash Sreekumar** — cross-platform full-stack developer (React, Node.js, Flutter, Android, on-device LLMs).

Live: <https://avisre.github.io/>

## Stack

Plain `index.html` + `style.css` + a tiny `script.js`. No build step, no frameworks, no trackers.

## Local preview

Open `index.html` in a browser, or run any static server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Pushes to `main` are auto-published by GitHub Pages.
